    Quick Reference¶

    Find the right service example quickly based on what you want to build.

    By Use Case¶

    "I want to build my first service"¶

    → Python Lambda Example - The simplest possible service → Python Service Example - Basic SDK usage

    "I need to build a REST API service"¶

    → FastAPI Service Template - REST API with FastAPI

    "I want to handle artifacts/files"¶

    → Python Service Example - File upload/download patterns → DataFabric Example - Collection management

    "I need asynchronous/queue processing"¶

    → Queue Example - Queue service patterns

    "I want to work with collections"¶

    → Collection Example - Collection parameter handling → DataFabric Example - DataFabric backend

    "I want to add AI/LLM capabilities"¶

    → AI Tool Template - Build reusable AI tools → Agent Tutorial - Multi-agent communication

    "I need multi-agent orchestration"¶

    → CrewAI Service - CrewAI integration → LlamaIndex Agent Runner - RAG and agents

    "I want to process documents for AI"¶

    → Markdown Conversion Service - Convert formats to markdown

    By Service Pattern¶

    Basic Service Structure¶

    • Python Lambda Example - Minimal service (best starting point)
    • Python Service Example - Standard service with artifacts
    • FastAPI Service Template - REST API service

    Data Handling¶

    • Collection Example - Collection parameters
    • DataFabric Example - Collection manager
    • Queue Example - Async processing

    AI & Agent Services¶

    • AI Tool Template - Reusable AI tool
    • Agent Tutorial - Multi-agent patterns
    • CrewAI Service - Multi-agent orchestration
    • LlamaIndex Agent Runner - RAG workflows
    • Markdown Conversion - Document processing

    Domain-Specific Examples¶

    • Gene Ontology Mapper - Bioinformatics tool
    • PaddlePaddle Segmentation - Deep learning service

    By Complexity¶

    Beginner (15-30 minutes each)¶

    Perfect for learning IVCAP fundamentals: - Python Lambda Example - Simplest service - Python Service Example - Basic patterns - Queue Example - Async basics

    Intermediate (1-2 hours)¶

    For developers building real services: - FastAPI Service Template - REST APIs - Collection Example - Collection handling - DataFabric Example - Collection management - AI Tool Template - AI tool creation

    Advanced (2-4 hours)¶

    Complex patterns and orchestration: - CrewAI Service - Multi-agent systems - LlamaIndex Agent Runner - RAG and agents - Agent Tutorial - Agent communication

    Decision Tree¶

    Start Here
    │
    ├─ Never used IVCAP?
    │   └─> Python Lambda Example → Python Service Example
    │
    ├─ Building a REST API?
    │   └─> FastAPI Service Template
    │
    ├─ Working with files/artifacts?
    │   └─> Python Service Example → Collection Example
    │
    ├─ Need asynchronous processing?
    │   └─> Queue Example
    │
    ├─ Building AI/LLM features?
    │   └─> AI Tool Template → Agent Tutorial
    │
    ├─ Need multi-agent orchestration?
    │   └─> CrewAI Service or LlamaIndex Agent Runner
    │
    └─ Domain-specific (bioinformatics, ML)?
        └─> Gene Ontology Mapper or PaddlePaddle Segmentation
