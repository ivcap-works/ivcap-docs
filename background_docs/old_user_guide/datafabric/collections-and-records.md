# Collections & records

This page will describe how IVCAP represents domain entities in the Data Fabric.

From the platform overview material, examples include:

- **Field trips**
- **Cameras**
- **Annotators**

The intent is to:

- group related records into **collections**
- attach metadata and relationships
- enable discovery and reuse across teams
