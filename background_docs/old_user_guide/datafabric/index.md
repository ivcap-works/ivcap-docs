# Data Fabric

The **Data Fabric** is IVCAP’s layer for organising and relating data assets so they can be **discovered**, **reused**, and **governed**.

The Jan 2024 overview frames the data fabric around concepts like:

- records and collections (e.g. field trips, cameras)
- annotations and datasets
- models and inference outputs
- observations/sightings
