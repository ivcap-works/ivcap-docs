## Prerequisites

All tutorials assume you have:

- `ivcap` CLI installed and authenticated ([install guide](install.md))
- Docker installed (for service-building tutorials)
- Python 3.11+ (for service-building tutorials)
- Access to an IVCAP deployment (ask your platform admin for the base URL)
