# User Guide

This guide is for **users of IVCAP** who want to:

- discover and run services
- monitor orders
- download and reuse artifacts
- explore and organise data using the **Data Fabric**

If you are new, start with **[Get started → Quick Start](../getting-started/quick-start.md)**.

## Typical user workflow

1. Authenticate to an IVCAP deployment
2. Discover a service
3. Create an order with parameters
4. Monitor until completion
5. Retrieve produced artifacts
6. (Optional) Register/tag outputs in the Data Fabric for later discovery and reuse
