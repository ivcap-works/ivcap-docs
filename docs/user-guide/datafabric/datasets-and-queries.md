# Datasets & queries ("dataset as query")

IVCAP aims to support **dataset as query**: datasets defined by intent (queries), not only by static file lists.

Example intent (from the platform overview):

> “A random sample of max. N images taken by camera C in geo-area G.”

## Why this matters

- Makes dataset definitions **repeatable**
- Enables sampling and re-materialisation as new data arrives
- Supports governance and auditability around “what data did we use?”
