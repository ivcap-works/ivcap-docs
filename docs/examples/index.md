## Learning Paths

### 🌱 Complete Beginner (2 hours)
1. Read the [Quick Start Guide](../getting-started/quick-start.md)
2. Follow the [Beginner Examples](beginner/index.md) path
3. Deploy your first service with [Python Service Example](https://github.com/ivcap-works/ivcap-python-service-example)

### 🔨 Tool Developer (1 hour)
1. Start with [Lambda Example](https://github.com/ivcap-works/ivcap-python-lambda-example)
2. Explore [FastAPI Template](https://github.com/ivcap-works/ivcap-python-fastapi-service-template)
3. Check the [Developer Guide](../developer-guide/README.md)

### 🤖 AI/ML Developer (2 hours)
1. Review [AI Tool Template](https://github.com/ivcap-works/ivcap-python-ai-tool-template)
2. Try [Agent Calling Agent Tutorial](https://github.com/ivcap-works/agent-calling-agent-tutorial)
3. Explore [CrewAI Service](https://github.com/ivcap-works/ivcap-crewai-service) for multi-agent systems

### 🧬 Bioinformatics Researcher (1 hour)
1. Examine [Gene Ontology Term Mapper](https://github.com/ivcap-works/gene-onology-term-mapper)
2. Adapt patterns to your domain-specific needs
3. Review [Service SDK documentation](https://github.com/ivcap-works/ivcap-service-sdk-python)
